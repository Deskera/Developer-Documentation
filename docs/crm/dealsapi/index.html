Deal APIs

This APIs will help you to manage your deals and pipelines records.

You will be able to do the following:

  • Deal:
    • Create, Retrieve, Update and Delete
  • Pipeline:
    • Create, Retrieve, Update and Delete
  • Stage:
    • Delete
